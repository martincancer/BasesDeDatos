Creación de Bases
y Tablas en SSMS

Paso a paso para crear una base de datos y una tabla usando la interfaz gráfica de SQL Server Management Studio, con capturas de referencia para acompañar la práctica.

Crear una Base de Datos

Desde el Explorador de objetos de SSMS se puede crear una base nueva usando el menú contextual, sin necesidad de escribir instrucciones SQL.

Primero hay que estar conectado a una instancia del Motor de base de datos de SQL Server. Luego se trabaja sobre el nodo Bases de datos.
Secuencia general
  • 1
    En el Explorador de objetos, expandir la instancia de SQL Server.
  • 2
    Hacer clic derecho sobre Bases de datos.
  • 3
    Elegir la opción Nueva base de datos.
  • 4
    Escribir el nombre de la base y aceptar.

Crear una Nueva Tabla

Una vez creada la base, el siguiente paso es ir al nodo Tablas y abrir el diseñador de tabla.

Primero expandí la base que acabás de crear, luego expandí Tablas.
Desde el Explorador de objetos
  • 1
    Expandir la base de datos creada.
  • 2
    Hacer clic derecho sobre el nodo Tablas.
  • 3
    Elegir Nueva y luego Tabla....

Definir Columnas y Tipos de Datos

En el diseñador de tablas se escriben los nombres de columna, se elige el tipo de dato y se indica si cada campo permitirá valores NULL.

Qué hay que completar
  • 1
    Nombre de columna.
  • 2
    Tipo de datos adecuado para cada campo.
  • 3
    Definir si se permiten o no valores nulos.
  • 4
    Si hace falta, usar el panel inferior de propiedades para configurar opciones adicionales.

Identidad y Clave Primaria

Es muy común que una columna entera sea autonumérica y funcione como clave principal de la tabla.

Para que una columna sea autonumérica, primero debe ser de tipo INT. Luego, en las propiedades de columna, se habilita la Especificación de identidad poniendo (Identidad) = Sí.
Pasos recomendados
  • 1
    Seleccionar la columna de tipo entero.
  • 2
    En Especificación de identidad, indicar Sí.
  • 3
    Hacer clic derecho sobre la columna y elegir Establecer clave principal.

Guardar la Tabla

Cuando la estructura ya está definida, se guarda la tabla desde el menú Archivo y se indica el nombre final.

Este paso crea definitivamente la tabla dentro de la base elegida.
Cierre del proceso
  • 1
    Ir a Archivo > Guardar tabla.
  • 2
    Escribir el nombre de la tabla.
  • 3
    Confirmar con Aceptar.
