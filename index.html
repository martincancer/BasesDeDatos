Disponible Clase 01 Fundamentos

Introducción a Bases de Datos

Primera parte de la materia: concepto de base de datos, tipos de bases de datos y explicación del SGBD como software encargado de administrar la información.

  • Definición de base de datos y diferencia entre dato, información y sistema.
  • Modelos relacionales y no relacionales, con sus contextos de uso.
  • Propiedades ACID y concepto de transacción.
  • Funciones principales de un SGBD y ejemplos de uso.

Disponible Clase 02 Diseño

Tablas, Relaciones y Normalización

Segunda parte del recorrido: cómo se estructuran las tablas, qué papel tienen las claves, de qué forma se relacionan las entidades y por qué normalizar mejora el diseño.

  • Filas, columnas, clave primaria y clave foránea.
  • Relaciones 1:N, 1:1 y N:M con ejemplos visuales.
  • 1FN, 2FN y 3FN con la tabla interactiva paso a paso.

Disponible Clase 03 SQL Server

Tipos de Datos en SQL Server

Una clase orientada a reconocer y elegir los tipos de datos más comunes en SQL Server: numéricos, fecha y hora, cadenas de caracteres, valores lógicos y datos binarios.

  • INT, DECIMAL, NUMERIC y FLOAT.
  • DATE, TIME, DATETIME, DATETIME2 y DATETIMEOFFSET.
  • CHAR, VARCHAR, NCHAR y NVARCHAR.
  • BIT, BINARY y VARBINARY.

Disponible Clase 04 SSMS

Creación de Bases y Tablas en SSMS

Una guía paso a paso para crear una base de datos y una tabla desde la interfaz gráfica de SQL Server Management Studio, con capturas de referencia para seguir el procedimiento.

  • Crear una base de datos nueva desde el Explorador de objetos.
  • Abrir el diseñador de tablas y definir columnas.
  • Configurar identidad autonumérica y clave primaria.
  • Guardar la tabla con nombre definitivo.
